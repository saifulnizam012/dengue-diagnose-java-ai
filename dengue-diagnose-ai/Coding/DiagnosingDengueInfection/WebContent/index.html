What Is Dengue?

Dengue fever is an infectious disease. It can cause high fevers, headaches, rashes, and pain throughout the body. Dengue viruses are spread to people through the bite of an infected Aedes species such as aegypti or albopictus mosquito. Almost half of the worldâ€™s population, about 4 billion people, live in areas with a risk of dengue.

Some symptoms related to dengue disease are:

  • Severe Headache
  • High Body Temperature
  • Abdominal Pain
  • Joint Muscle Ache

Process

How The Intelligent System Works

1. Training

The machine learns about the dengue disease by developing a predictive model based on both input and output data provided by the engineer known as 'The Train Set'.

2. Testing

The machine classifies both input and output data of 'The Test Set' to validate the model built during the training phase.

3. New Data

The machine receives a new data from the user input after it passes both Training and Testing phase.

4. Prediction

The machine predicts the output of the newly received data from the user input.







User Guide

A Little Tip On How To Use The System

Step 1

Click the 'Intelligent System' at the navigation bar, it will navigate you to the 'Take The Test' button. Click the button to start diagnosing the symptoms.

Step 2

For admin, click 'Train' and 'Test' button before you start filling in the forms with the required data about yourself and the symptoms that you have been experiencing.

Step 3

Click 'Evaluate' to diagnose your symptoms. The intelligent system will display the dengue test result according the symptoms entered.
